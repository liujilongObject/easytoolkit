# EasyWebToolkit

> 一个**纯前端、免上传**的在线文件处理工具站。
> 你的文件不会离开浏览器——零服务器、零账号、零追踪。

🌐 **在线体验：** https://liujilongobject.github.io/easytoolkit/

---

## 这是什么？

EasyWebToolkit 是一款运行在浏览器里的**轻量级文件处理工具集**。
所有计算都通过 WebAssembly 在你本地完成，不需要登录、不需要安装客户端、不需要把文件传到任何服务器。

**打开网页 → 拖入文件 → 处理 → 下载**，就这么简单。

---

## 它能帮你做什么？

### 🖼️ 图片工具

| 功能 | 说明 |
| ---- | ---- |
| **格式转换** | PNG / JPG / WebP 三向互转，质量参数可调 |
| **智能压缩** | 选择压缩级别，算法自动搜索最佳质量与体积平衡点。JPG 通过二分搜索 quality 参数；PNG 支持无损优化与颜色量化两种策略 |
| **裁剪** | 多手柄拖拽 + 长宽比锁定（自由 / 1:1 / 4:3 / 3:2 / 16:9 / 9:16） |
| **尺寸调整** | 像素 / 百分比双模式，长宽比锁定，多种重采样滤镜（Lanczos3 / CatmullRom 等） |
| **亮度调整** | 滑块控制 -100 ~ +100 亮度增量，逐像素处理后以源格式输出 |
| **图片滤镜** | 灰度、高斯模糊、快速模糊、锐化、反色、对比度、色相旋转、复古（棕褐色）等滤镜，支持自定义强度参数 |
| **图片水印** | 单点定位（9 锚点 + 自定义坐标）和平铺模式，支持透明度、缩放、旋转参数 |
| **GIF 压缩** | 对静态/动画 GIF 执行帧间差异编码 + 调色板量化压缩，三档压缩等级（轻度 / 中度 / 重度），保留动画元数据 |
| **多图合成 GIF** | 将多张 PNG / JPG / WebP / GIF 图片合成为动画 GIF，可配置帧延迟、画布尺寸、调色板颜色数 |
| **GIF 逐帧分解** | 一键拆分动画 GIF 为逐帧无损 PNG，保持原始分辨率与画质，可预览每帧并单张或 ZIP 批量下载 |

### 📄 PDF 工具（开发中）

PDF 压缩、PDF ↔ 图片互转、合并 / 拆分等能力规划中。

## 为什么选它？

### 🔒 真正的隐私安全

- **文件不离开浏览器**：所有处理在你本机的浏览器里完成，没有任何文件、内容或元数据被发送到服务器。
- **零账号、零跟踪**：无需注册登录，不接入第三方统计/监控 SDK。
- **可离线使用**：通过 PWA 缓存机制，加载一次后即使断网也能继续使用（规划中）。

> 适合处理**机密合同、个人证件、内部资料**等任何不便上传到第三方网站的文件。

### ⚡ 高性能 · 不卡顿

- **WebAssembly 原生级速度**：图片编解码核心由 Rust 编译为 wasm，性能接近本地软件。
- **Web Worker 后台计算**：所有重活都在子线程跑，UI 永远流畅，不会让你的页面卡死。
- **路由级懒加载**：进入对应工具页才加载相关 wasm 模块，首屏极快（首屏 JS gzip 仅 ~38 KB）。

### 🎯 简单易用

- 拖拽上传、即时预览、一键下载。
- 操作路径短，参数默认值经过精心调优。
- 转换前后体积对比一目了然。
